# 📊 Marketing Campaign Performance Analysis

## 📌 Project Overview
This project analyzes marketing campaign performance using Python.  
The dataset contains 22,000+ campaign records with metrics like ROI, Conversion Rate, Engagement Score, and Acquisition Cost.

The goal is to extract insights and visualize campaign effectiveness across different channels, companies, customer segments, and locations.

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset
Dataset Source:
https://raw.githubusercontent.com/ArchanaInsights/Datasets/main/marketing_campaign.csv

Total Records: 22,029  
Total Columns: 16  

---

## 📊 Key Analysis Performed

- Campaign Type Distribution
- Channel Performance Analysis
- ROI vs Acquisition Cost
- Engagement Score Analysis
- Conversion Rate by Location
- Customer Segment Performance
- Time Series Trend Analysis
- Correlation Heatmap

---

## 📈 Key Insights

- ROI varies significantly across campaign types.
- Social Media and Search campaigns show strong engagement.
- Acquisition cost impacts ROI performance.
- Certain locations generate higher conversion rates.
- Engagement score positively correlates with conversion rate.
